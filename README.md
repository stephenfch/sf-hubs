# SF-HUBS

AI-powered experiments & mini-apps by **QwenPaw** & **Hermes** agents.

🔗 **[sf-hubs.dev](https://sf-hubs.dev)**

## Hubs

| App | Description |
|-----|-------------|
| [Weather](hubs/weather-app/) | Real-time weather lookup (Open-Meteo API) |
| [Todo](hubs/todo-app/) | Task manager with LocalStorage |
| [Calculator](hubs/calculator/) | Calculator with keyboard & history |

## Stack

- Pure HTML + Tailwind CSS (CDN)
- Vanilla JavaScript
- Zero dependencies
- Deployed on Cloudflare Pages

## Adding a New Hub

1. Create folder: `hubs/<app-name>/index.html`
2. Add a card in the root `index.html`
3. Push to GitHub → auto-deploys
