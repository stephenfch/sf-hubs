📖 P2 教材 · 閱讀理解・寫作

📖 閱讀理解

📖 我的小狗 ▶
📖 短文：我的小狗

我家有一隻小狗，名叫「毛毛」。毛毛全身都是白色的毛。每天我放學回家，毛毛都會搖着尾巴歡迎我。

毛毛是黑色的
毛毛的毛是白色的 ✓
毛毛喜歡睡覺
✅ 正確！「毛毛全身都是白色的毛」。

✍️ 寫作技巧

📚 看圖寫段 ▶
  1. 👀 看圖：人物、地點、事件
  2. 🤔 想前因後果
  3. 📝 組織：開頭→中間→結尾
  4. ✍️ 用連接詞串連

連接詞： 首先 然後 最後 因為 所以

📖 難忘的一天

上星期六，爸爸帶我去海邊游泳。我第一次看見大海，覺得很興奮。我和爸爸一起堆沙堡。回家的時候，我依依不捨。

作者去山上遊玩
作者和爸爸去海邊 ✓
作者覺得大海很無聊
✅ 正確！「爸爸帶我去海邊」。

📖 我的學校

我的學校很大很漂亮。校門口有一棵大榕樹。操場上有鞦韆和滑梯。我最喜歡圖書館，因為裡面有很多有趣的書。

學校很小
校門口有大榕樹 ✓
作者不喜歡上學
✅ 正確！「校門口有一棵大榕樹」。

進階：

作者最喜歡操場
作者最喜歡圖書館 ✓
作者最喜歡課室
✅ 正確！「我最喜歡圖書館」。
📋 教案

📖 教案一：P1 聆聽訓練

🎯 目標學生能聽懂短篇故事，並回答相關問題
⏱️ 時間35分鐘
📝 流程引入（5分鐘）→ 聆聽故事兩遍（10分鐘）→ 分組討論（10分鐘）→ 工作紙（10分鐘）

📖 教案二：P2 閱讀理解

🎯 目標學生能理解短文內容，回答事實性和推斷性問題
⏱️ 時間40分鐘
📝 流程引入（5分鐘）→ 默讀短文（10分鐘）→ 逐段提問（15分鐘）→ 進階問題（10分鐘）

📖 教案三：針對性 — 配詞填充訓練

🎯 目標學生能正確選擇詞語配搭填空
⏱️ 時間30分鐘
📝 流程講解配搭（10分鐘）→ 示範解題（5分鐘）→ 獨立練習（10分鐘）→ 訂正（5分鐘）

📖 教案四：針對性 — 閱讀理解專項

🎯 目標學生能運用三步法解答閱讀理解題
⏱️ 時間40分鐘
📝 流程介紹三步法（10分鐘）→ 示範應用（10分鐘）→ 練習（15分鐘）→ 訂正討論（5分鐘）
